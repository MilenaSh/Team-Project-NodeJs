 +## Getting Started
 +
 +Install libraries on your machine
 +```
 +npm install
 +```
 +
 +Run the project
 +```
 +npm start
 +```
 +## Running the tests
 +
 +```
 +- browser -> gulp tests:browser
 +- integration -> gulp tests:integration
 +- unit -> gulp tests:unit
 +```
